
# DELIGHT SHOP

Este proyecto consta de un e-commerce de ropa realizado con React, aplicando buenas prácticas y usando FireBase para almacenar pedidos y stock.

## Inicializacion

Para inicializar este proyecto escribir en terminal:
```bash
  npm run start
```
